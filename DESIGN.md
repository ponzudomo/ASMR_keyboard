# Design Concept: Healing ASMR Keyboard

## 1. Vision & Core Value
- **Concept:** 「タイピングを、心を整える時間に変える」
- **Goal:** 音（ASMR）と視覚効果（テーマデザイン）が完全に同期し、触れるたびにストレスが解消されるような没入型の入力体験を提供すること。

## 2. Global Design Principles
- **Organic Motion:** すべてのアニメーションは機械的（Linear）ではなく、有機的（Spring/Ease）である。`Spring` や `Ease-In-Out` を多用し、生命感のある動きを追求。
- **Low Latency:** 打鍵音は指先の動きに即座に反応し、0.01秒の遅延も感じさせないこと。
- **Consistency:** 「音・背景・キーの形状・反応」の4要素が1つのテーマとして一貫性を持っていること。

## 3. Theme Variations (Sound & Visual Sets)

現在、以下の5つの主要テーマを開発の柱とする。
ここから増やしていく予定である

### A. Frozen Serenity: ひんやりと冷たい静的な氷デザイン
- **Description:** 氷をパリッと砕く快感を追求。
- **Sound (ASMR):** 硬い氷を削るようなクリスタルな音。高い音が打鍵音として鳴り続けると鬱陶しいので、低くくぐもった音にする。
- **Background:**
  - 深く青白いグラスモーフィズム。氷の壁越しに冷たい光が透過しているようなデザイン。 
  - 全体に軽く光沢がかかっている。
    - **Refreshing Fizz**: 透明感のある動的なデザイン
    - 全体が液体で満たされている表現。指定されたカラー（サイダー, メロン, コーラなど）の液体の中を、大小様々な気泡が下から上へゆっくりと有機的に昇っていく。
    - キーは液体の中に浮遊している氷のように見え、気泡がキーの裏側を通過したり、キーのエッジに当たって軌道を変えたりする（ように見えるレイヤー構造）。
- **Key Design:** 透明度の高い氷のブロック。エッジはほんのり丸みを持たせる。とにかく立体感と光沢感のあるリッチな見た目にしたい。
- **Interaction:**
  - 押下時にキーに「ヒビ」が入るような視覚効果。キーの真ん中から蜘蛛の巣状の細かいヒビがキー全体に広がる。
  - ヒビの入り方は複数種類用意し、ランダムに適用させることで単調さを防ぐ。
  - ヒビ割れと同時に、キーの周囲に微細な白い粉（氷の破片）と、ドライアイスのような白い煙が「フワッ」と広がり、すぐに消える。
  - 指を離してから約0.5秒〜1秒かけて、入ったヒビが「外側から中心に向かって」凍りつくように消えていく。
  - 単純なフェードアウトではなく、「水が氷に戻る」ような有機的な（Ease-In-Out）アニメーションにする。
  - これにより、ユーザーは次々にキーを「壊す」ことができ、画面がヒビだらけになるのを防ぎつつ、視覚的な満足感を持続させる。

### B. Refreshing Fizz: 透明感のある動的な氷デザイン
- **Description:** シュワシュワとした炭酸と氷の音で爽快感を追求。
- **Sound (ASMR):** 氷の「カラコロ」とした軽快な音 ＋ 背景音として微弱な炭酸の「シュワシュワ」ループ。
- **Background:**
  - 全体が液体で満たされている表現。指定されたカラー（cider, melon, colaなど）の液体の中を、大小様々な気泡が下から上へゆっくりと有機的に昇っていく。
  - キーは液体の中に浮遊している氷のように見え、気泡がキーの裏側を通過したり、キーのエッジに当たって軌道を変えたりする（ように見えるレイヤー構造）。
- **Key Design:** Frozen Serenityを流用し、色を調整。
- **Interaction:**
    - キーが押された反動で液体中を沈み込み、ゆったりと浮上する（Buoyancyエフェクト）。
    - 押したキーの揺れが隣接するキーにも伝搬する。

### C. Wood (木の温もり)
- **Description:** アナログで暖かい体験。音の癒し効果重視。
- **Sound (ASMR):** 木琴のようなコトコト、ポコっというやや低くくぐもった音の打鍵音。
- **Background:** 落ち着いた木目調のテクスチャ。
  - **Customization:** 数種類のウッドカラー（オーク、ウォルナット、チェリー等）から選択可能にする。
- **Key Design:** 木の質感を活かしたウッドブロック。
    - **Customization:** バックグラウンドと同様、ウッドカラーを選択可能にする。
- **Interaction:** 物理的なボタンのように「深く沈み込む」影の表現。

### D: Slime Pop（もちもちスライム）
- **Description:** 弾力と粘性。指に吸い付くような「もちっ」とした感触と、スライムを潰す快感。
- **Visuals:**
    - **Background:** 
      - パステルカラーのグラデーション（例：ラベンダー、ミントグリーン）。
      - 液体がゆっくりと波打つような「Liquid Background」演出。
- **Key Design:** 
  - **Neumorphism:** 背景と同じ色で、柔らかな影（Shadow）と光（Highlight）のみで形を浮き上がらせる。
  - エッジは極限まで丸くし、表面に「ツヤ」を持たせて水分量を感じさせる。
- **Interactions:**
    - **On Press (Deformation):** 
      - 押した瞬間に「パチン」と消えるのではなく、まずキーがぐにゃっと横に広がる（ScaleX: 1.1, ScaleY: 0.9）。
      - 粘り気（Viscosity）を表現するため、指の動きに対してわずかに遅れて変形する。
    - **On Pop:** 指を離した瞬間に、小さな気泡が飛び散るエフェクトと共に平らになる。
    - **Regeneration:** 0.8秒ほどかけて、下からゆっくりと空気が入るように元の「ぷっくり」した形に盛り上がる。
- **Sound (ASMR):** 
    - 「ムギュッ」「クポッ」という粘土の高いウェットな破裂音。
    - キーが復活する際に、微かな「シュポッ」という吸気音。

### E: Cozy Ember（夜の焚き火）
- **Description:** 暗闇の中の暖かさと安らぎ。薪がはぜる音と、ゆらめく火影。
- **Visuals:**
    - **Background:** 深いチャコールグレー（炭の色）の背景。
        - **Firelight Flicker:** 画面の端（あるいは下部）から、オレンジ色の暖かな光が「ゆらぎ」を持って差し込む演出。
        - 背景全体に、時折小さな「火の粉」が下から上へ舞い上がる。
    - **Key Design:**
      - 炭化した薪（Charred Wood）のような、マットで質感のあるテクスチャ。
      - 刻印（文字）は、火が灯っているようにボワっと光る暖かなオレンジ色。
- **Interactions:**
    - **On Press:** 
      - 押したキーが「パッ」と明るく赤熱し、周囲に火の粉（Sparks）が飛び散る。
      - まるで薪を突いて、火が勢いづくような演出。
    - **On Release:** 赤熱した色が、ゆっくりと（Ease-Out）元の暗い炭の色に戻っていく。
- **Sound (ASMR):** 
    - **BGM:** 背後で常に流れる、小さく落ち着いた「パチパチ…」という薪がはぜる環境音。
    - **Tap:** 乾いた木の「カラコロ」「コトッ」という、薪を火に投下した時のような硬質な打鍵音。
    - **Random Effect:** 数回に一度、打鍵音に同期して少し大きめの「パチンッ！」というはぜる音を混ぜる。

## 4. Technical Implementation Notes (for AI)
- **Layout:** Jetpack Compose を使用。`LazyVerticalGrid` またはカスタムレイアウトでキー配列を構築。
- **Theming:** - `KeyboardTheme` sealed class を定義し、テーマごとに `Colors`, `Shapes`, `Sounds` を切り替える設計にする。
- **Effects:**
    - グラスモーフィズムは `Modifier.blur()` および `GraphicsLayer` を活用。
    - 泡のはじける挙動や氷のヒビは、Canvasでの描画またはLottieアニメーションの利用を検討。
- **Audio:** - 全テーマ共通で `SoundPool` を使用。
    - 打鍵ごとに `setRate()` を用い、ピッチをランダムに微調整して「本物感」を出す。
