# ASMR Keyboard

## コンセプト

ASMRのような心地よいタイプ音とアニメーションが流れる癒し系キーボードアプリです。
毎日のタイピングを癒しの時間に変えることを目指します。

## 開発ロードマップ

### フェーズ1: 基本的なIMEの実装
- [ ] `InputMethodService` を継承した IME の基本構造を作成する
- [ ] シンプルなキーボードUI（アルファベット、数字、記号）を実装する
- [ ] 入力した文字がテキストフィールドに反映されるようにする

## 学習フロー

Android IME開発やJetpack Composeを使ったUI構築が初めての場合は、以下の順で学習を進めるのがおすすめです。

1.  **Kotlinの基礎:** Android開発の公式言語であるKotlinの文法や特徴を学びます。
2.  **Androidアプリ開発の基礎:** Activity、Intent、基本的なUIコンポーネントなど、通常のAndroidアプリ開発の知識を習得します。
3.  **Jetpack Compose:** 宣言的なUIツールキットであるJetpack Composeの基本的な使い方を学び、簡単なUIを作成してみます。
4.  **InputMethodService:** Android IMEの根幹となる `InputMethodService` の仕組みとライフサイクルを学びます。公式ドキュメントやサンプルコードを参考に、簡単なIMEを作成してみましょう。
5.  **カスタムUIとアニメーション:** Jetpack Composeを使って、独自のキーボードレイアウトやアニメーションを作成する方法を学びます。
