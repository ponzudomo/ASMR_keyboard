# ASMR Keyboard（仮）

## コンセプト

ASMRのような心地よい打鍵音と高品質なアニメーションを搭載した、華やかさ重視のキーボードアプリを目指して開発しています。
中高生向けのポップで可愛いキーボードアプリを今まで使っていたのですが、もう少し大人っぽい落ち着いたコンセプトのものが欲しくて作っています。
あのワクワク感はそのまま、ターゲット層を少し上に見据えたアプリを作るのが目標です。

## このリポジトリについて

ドキュメントは大体 `docs/` というフォルダにまとめるようにしていて、[そこのREADME](docs/README.md) を読むと概観が分かるようにしています。
リポジトリ構造などはまたこれからまとめます。
このファイルには以下プロジェクトのざっくりとした説明を書いておこうと思います。

### 2. 視差効果 (Parallax)
iOSの待ち受け画面の視差効果が面白かった。
画面の傾きに応じて待ち受け画面が立体的に動いて見えるようである。これをキーボードで実装できたら面白いんじゃないかと思った。
現時点で想定しているターゲット層はおそらく3DSとかにときめいていた世代なので、そういう意味でもちょうどいいんじゃないかと思う。

### 実装済み (Implemented)

- **基本的なキーボード機能:**
    - `InputMethodService` を利用したIMEの基本構造が完成し、キーボードが表示され、アルファベットの入力が可能。
- **UIとテーマシステム:**
    - UIはすべてJetpack Composeで構築。
    - `Wood`, `FrozenSerenity`, `SlimePop`, `CozyEmber` のカラーテーマを定義。（仮）
    - キーボード上部のボタンで、リアルタイムにテーマを切り替える機能を実装。
- **インタラクション:**
    - `Wood` テーマにおいて、`DESIGN.md` の仕様に基づき、キーが「深く沈み込む」アニメーションを実装。
