# ASMR Keyboard

## コンセプト

ASMRのような心地よい打鍵音と高品質なアニメーションを搭載した、癒し系キーボードアプリを目指して開発中です。
日々の入力体験をより贅沢なものにしてくれる、そんなアプリケーションを目指しています。

## 現在の状況 (Current Status)

### 実装済み (Implemented)

- **基本的なキーボード機能:**
    - `InputMethodService` を利用したIMEの基本構造が完成し、キーボードが表示され、アルファベットの入力が可能。
- **UIとテーマシステム:**
    - UIはすべてJetpack Composeで構築。
    - `Wood`, `FrozenSerenity`, `SlimePop`, `CozyEmber` のカラーテーマを定義。（仮）
    - キーボード上部のボタンで、リアルタイムにテーマを切り替える機能を実装。
- **インタラクション:**
    - `Wood` テーマにおいて、`DESIGN.md` の仕様に基づき、キーが「深く沈み込む」アニメーションを実装。

### 未実装 (Pending)

- **打鍵音:** 現在サウンドファイルを選定中のため、`SoundPool` の実装は保留中。
- **テーマ別インタラクション:** ユニークな視覚効果（氷のヒビ、スライムの変形など）は未実装。

### Phase 1: キーボードコア機能の強化 (必須)

- [ ] **Backspaceキー:** 文字を削除する機能。
- [ ] **Spaceキー:** スペースを入力する機能。
- [ ] **Shiftキー:** 大文字と小文字を切り替える機能。
- [ ] **Enterキー:** 改行や送信を行う機能。
- [ ] **数字・記号レイアウト:** 数字や基本的な記号を入力可能にする。

### Phase 2: ASMR体験の深化

- [ ] **打鍵音の実装:** `SoundPool` を利用し、低遅延で高品質なサウンドを再生する。
- [ ] **テーマ別インタラクション:** `DESIGN.md` に基づき、各テーマ固有のアニメーションを実装する。
    - [ ] **Frozen Serenity:** ヒビ割れエフェクト
    - [ ] **Slime Pop:** スライム変形エフェクト
    - [ ] **Cozy Ember:** 火の粉エフェクト

### Phase 3: 設定と永続化

- [ ] **テーマ選択の永続化:** ユーザーが選択したテーマをデバイスに保存し、次回起動時にも適用されるようにする。
- [ ] **設定画面:** アプリ内に専用の設定画面を作成する。
