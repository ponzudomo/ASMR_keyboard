# Theme: Slime Pop (もちもちスライム)

## 1. コンセプト
指に吸い付くような「もちっ」とした感触と、スライムを潰す快感を視覚と音で再現します。

## 2. ビジュアル (Visuals)
- **Background:** パステルカラーのグラデーション（ラベンダー、ミントグリーン）。液体がゆっくりと波打つような「Liquid Background」演出。
- **Key Design:** ニューモーフィズムを採用。背景と同じ色で、柔らかな影と光のみで形を浮き上がらせます。表面に「ツヤ」を持たせて水分量を表現。
- **Parallax:** 
  - Background (L1): 波打ちに合わせてゆらゆらと動く。
  - Key Surface (L3): 表面のハイライトが傾きに合わせて移動し、立体的な光沢を演出。

## 3. インタラクション (Interactions)
- **On Press:** 
  - キーが「ぐにゃっ」と横に広がる（ScaleX: 1.1, ScaleY: 0.9）。指の動きに対してわずかに遅れて変形する粘性（Viscosity）を表現。
- **On Release:** 
  - 小さな気泡が飛び散るエフェクトと共に、0.8秒ほどかけて下からゆっくりと空気が入るように元の「ぷっくり」した形に盛り上がる。

## 4. サウンド (ASMR)
- **Tap:** 「ムギュッ」「クポッ」という粘性の高いウェットな破裂音。
- **Regeneration:** キーが復活する際に、微かな「シュポッ」という吸気音。
