# Theme: Cozy Ember (夜の焚き火)

## 1. コンセプト
暗闇の中の暖かさと安らぎ、薪がはぜる音とゆらめく火影を再現した、リラックス重視のテーマです。

## 2. ビジュアル (Visuals)
- **Background:** 深いチャコールグレー（炭の色）。画面下部からオレンジ色の暖かな光が「ゆらぎ」を持って差し込む演出（Firelight Flicker）。時折、小さな「火の粉」が下から上へ舞い上がる。
- **Key Design:** 炭化した薪（Charred Wood）のような、マットで質感のあるテクスチャ。刻印（文字）は、火が灯っているようにボワっと光るオレンジ色。
- **Parallax:** 
  - Background (L1): 背景の火影が、傾きに応じてわずかに揺らぐ。
  - Overlay (L5): 前面を舞う火の粉が、視差効果によって浮いているように見える。

## 3. インタラクション (Interactions)
- **On Press:** 
  - 押したキーが「パッ」と明るく赤熱し、周囲に火の粉（Sparks）が飛び散る。薪を突いて火が勢いづくような演出。
- **On Release:** 
  - 赤熱した色が、ゆっくりと（Ease-Out）元の暗い炭の色に戻っていく。

## 4. サウンド (ASMR)
- **BGM (Environment):** 背後で常に流れる、小さく落ち着いた「パチパチ…」という薪がはぜる環境音。
- **Tap:** 乾いた木の「カラコロ」「コトッ」という硬質な打鍵音。
- **Randomness:** 数回に一度、打鍵音に同期して少し大きめの「パチンッ！」というはぜる音を混ぜる。
