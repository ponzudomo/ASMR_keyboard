# Theme: Frozen Serenity (氷の静寂)

## 1. コンセプト
「氷をパリッと砕く快感」と「冷たく澄んだ空気感」を追求したテーマです。

## 2. ビジュアル (Visuals)
- **Background:** 深い青白いグラスモーフィズム。氷の壁越しに冷たい光が透過しているようなデザイン。
- **Key Design:** 透明度の高い氷のブロック。エッジに微小なハイライト（0.5dp White）を入れ、鋭利さを表現。
- **Parallax:** 
  - Background (L1): 最も奥深く。
  - Key Base (L2): 奥行き感を強調する濃い影。
  - Overlay (L5): 画面全体に薄い霧のようなグラデーション。

## 3. インタラクション (Interactions)
- **On Press:** 
  - キーにランダムな「ヒビ」のテクスチャが表示される。
  - 微細な「氷の破片（パーティクル）」が周囲に飛び散る。
- **On Release:** 
  - 0.5秒〜1秒かけて、ヒビが外側から中心に向かって凍りつくように消えていく（Ease-In-Out）。

## 4. サウンド (ASMR)
- **Tap:** クリスタルな高い音。ただし、不快にならないよう低域を強調し、くぐもった音色にする。
- **Release:** 氷が再結晶化するような、微かな「チリッ」という音。
