# Theme: Wood (木の温もり)

## 1. コンセプト
「アナログな木の質感」と「心地よい打鍵音」による、温かく落ち着いたタイピング体験を提供します。

## 2. ビジュアル (Visuals)
- **Background:** 落ち着いた木目調のテクスチャ。ウォルナットやオークなど、複数のバリエーションを用意。
- **Key Design:** 木の質感を活かしたウッドブロック。表面はマットな質感に。
- **Parallax:** 
  - Background (L1): 傾きに応じてゆっくりと動き、木目の奥行きを強調。
  - Key Base (L2): 光源が固定されているように、影のオフセットを動的に変更。

## 3. インタラクション (Interactions)
- **On Press:** 
  - 物理的なボタンのように「深く沈み込む」アニメーション。`spring`の`stiffness`は`MediumLow`に設定し、重厚感を演出。
- **On Release:** 
  - ゆっくりと元の位置に復帰。動きの終わりにわずかなバウンド（弾み）を加える。

## 4. サウンド (ASMR)
- **Tap:** 木琴のような「コトコト」「ポコッ」という、やや低くくぐもった音。
- **Randomness:** 複数の打鍵音を用意し、ピッチをわずかにランダム化することで、単調さをなくします。
